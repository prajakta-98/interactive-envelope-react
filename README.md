# 💌 React Envelope Animation

An interactive envelope animation built using React and Framer Motion.

Hover over the envelope to reveal a letter with floating hearts and smooth transitions.

---

## ✨ Features

- Smooth hover animation
- Floating heart micro-interactions
- Clean component structure
- Fully responsive layout
- Modern UI styling

---

## 🛠 Tech Stack
